LogiNext Delivery Assignment System

Description:
This Java application assigns delivery drivers to customer orders
based on order placement time and driver availability.

Logic:
- Each driver can handle only one order at a time.
- The lowest indexed available driver is assigned first.
- If all drivers are busy, the system prints "No Food :-(".

How to Run:
1. Compile: javac App.java
2. Run: java App
3. Enter input as:
   N M
   O1 T1
   O2 T2
   ...
